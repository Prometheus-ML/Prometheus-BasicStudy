## 1. File Structure
```bash
root
├── materials
│   ├── wk1
│   ├── wk2
│   ├── ...
├── teams
│   ├── team1
│   │   ├── wk1
│   │   ├── wk2
│   │   ├── ...
│   ├── team2
│   │   ├── ...
│   ├── team3
│   │   ├── ...
```
- ```materials``` : 스터디 자료를 모은 폴더
- ```teams``` : 팀별로 폴더를 만들어서 자료를 정리한 폴더
    - ```wk1```, ```wk2```, ... : 각 주차별로 수행한 자료를 정리한 폴더

**팀별 자료 정리 규칙**
```bash
1. wk$ 에는 $주차에 진행한 자료, 코드를 넣습니다.
2. 해당주차 코드를 넣을 때, 이름은 "assignment_wk$_(이름).ipynb" 형태로 합니다.
```
